# Telegram Bot для кафе

Бот автоматизирует:
- Заказы еды
- Бронирование столов
- Сбор отзывов
- Интеграцию с Google Sheets

## Технологии
- Python 3.13
- Aiogram
- Google Sheets API
- Git + GitHub

## Автор
@KaynarKafe_bot — Telegram-бот для кафе «Кайнар»
